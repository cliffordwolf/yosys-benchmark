# bft-flow
Butterfly Fat Tree NoC enhanced with in-order routing capability supported by lightweight flow control and pruning of switch crossbar. 
RTL for the entire NoC with a simulation and implementation framework targeting Xilinx FPGAs. 
FCCM 2019 Artifact.

[Reference](https://git.uwaterloo.ca/watcag-public/bft-flow)

## Changes for Benchmarks
  * Generated top level file for each of the 4 supported topologies.
  * System size of the NoC: 128.
  * Data width: 32.
